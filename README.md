# MovieBrowser – Projet React OFPPT

Ce projet a été réalisé dans le cadre du module Développement Front-end.

C’est une application web faite avec React qui permet d’afficher une liste de films en utilisant l’API TMDB (The Movie Database).  
On peut consulter les détails d’un film, ajouter des films aux favoris et marquer les films déjà vus.

## Fonctionnalités

- Affichage des films populaires
- Page avec les détails d’un film
- Ajout et suppression des favoris
- Gestion des films déjà vus
- Sauvegarde des données dans le navigateur
- Formulaire simple avec validation
- Gestion du chargement et des erreurs

## Technologies utilisées

- React
- React Router
- Redux Toolkit
- API TMDB

## Lancer le projet

1. Télécharger le projet
2. Installer les dépendances :

   npm install

3. Lancer le serveur :

   npm run dev

Puis ouvrir le lien affiché dans le navigateur.
